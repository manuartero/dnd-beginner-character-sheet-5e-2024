# Icon Assets Index

SVG icon library organized in 10 volumes. Each volume contains **BLACK** and **WHITE** variants.

All SVGs use a 100x100 viewBox with single-color fills (`#00040c`). In the app, icons are rendered via CSS `mask-image` with `background-color: var(--shade-3)` for theming.

### Other assets in this folder

- `ddb-icons/schools/` — 8 spell school PNGs (abjuration, conjuration, divination, enchantment, evocation, illusion, necromancy, transmutation). **Currently unused** in the codebase.
